# <a name="set-field-values-using-parameters-passed-to-a-form---lookup-fields"></a>Nastavení hodnot polí pomocí parametrů předaných formuláři - vyhledávacím polím

Nastavte hodnotu vyhledávacích polí a příklad: Pomocí Xrm.Utility.openEntityForm otevřete nové okno. Nelze nastavit hodnoty pro partylist nebo pokud jde o vyhledávání.

Další informace naleznete v [tématu Nastavení hodnot pro vyhledávací pole](https://docs.microsoft.com/previous-versions/dynamicscrm-2016/developers-guide/gg334375(v=crm.8)#set-values-for-lookup-fields).
