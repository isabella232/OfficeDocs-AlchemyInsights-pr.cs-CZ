# <a name="azure-active-directory-security-and-identity"></a>Azure Active Directory (zabezpečení a identita)

Vyberte jeden z následujících průvodců nastavením:

- **Uživatelé** služby Azure Active Directory Connect [Sync](https://go.microsoft.com/fwlink/?linkid=2071310) do vašeho adresáře Nakonfigurujte synchronizaci hodnot hash hesel nebo předplatné ověřování, aby se uživatelé mohli přihlásit ke svému e-mailu a k místní službě Active Directory (řadič domény) pomocí stejného hesla. Můžete taky nastavit přihlašování federace se službou AD FS (Active Directory Federation Services).

- **Nastavení funkcí Azure** AD (Azure [AD)](https://go.microsoft.com/fwlink/?linkid=2134390) Nastavte pokročilé funkce dostupné v Azure Active Directory Premium, jako je vícefaktorové ověřování, registrace zařízení a monitorování zabezpečení.
