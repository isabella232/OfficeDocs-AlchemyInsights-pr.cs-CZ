# <a name="give-users-access-to-sharepoint-site"></a>Přístup uživatelům k SharePoint webu

Funkce žádosti o přístup umožňuje uživatelům požádat o přístup k obsahu, který momentálně nemají oprávnění k zobrazení. Jako vlastník webu můžete nakonfigurovat funkci tak, aby vám posílala poštu, když někdo požádá o přístup k webu. Další informace najdete na stránce Nastavení [a správa žádostí o přístup.](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3)

Tady jsou některé běžné odkazy týkající se žádostí o přístup:

- [Access Denied" to Access Requests list or "Request approval failed" when you process a pending request in SharePoint Online](https://docs.microsoft.com/sharepoint/support/sharing-and-permissions/request-approval-failed)

- [Změna nastavení žádostí o přístup](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [Schválení nebo odmítnutí žádostí o přístup k webu](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [Správa pozvánek pro externí uživatele (SharePoint Online)](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [Opětovné poslat pozvánku](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)
