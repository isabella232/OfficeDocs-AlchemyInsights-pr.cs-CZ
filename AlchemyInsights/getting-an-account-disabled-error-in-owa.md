# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>V Outlooku na webu se zobrazuje chyba neaktivního účtu?

Pokud se zobrazí chybová zpráva, že byl **váš účet zakázán**  `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` při pokusu o otevření Outlooku na webu (dříve známý jako Outlook Web App), mohl správce organizace zakázat přístup k Outlooku na webu. Správce může tuto chybu opravit podle pokynů v tématu [Povolení nebo zakázání Outlook Web Appu pro poštovní schránku](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
